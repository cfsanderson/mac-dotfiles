# mac-dotfiles

macOS dotfiles for my M1 MacBook Pro, managed with [GNU Stow](https://www.gnu.org/software/stow/).

Companion to [asahi-dotfiles](https://github.com/cfsanderson/asahi-dotfiles) (Hyprland on Fedora Asahi) and [arch-dotfiles](https://github.com/cfsanderson/arch-dotfiles). Shares the same color scheme, Star Wars ASCII art gallery, and most tool configs across all three machines.

---

## Stow Packages

| Package | Contents |
|---|---|
| `bat` | Syntax highlighter config + cfs-gruvbox-material theme |
| `brew` | Brewfile for reproducible package installs |
| `btop` | System monitor config + cfs-gruvbox-material theme |
| `colors` | Master gruvbox-material color palette (CSS, TOML, shell, HTML) |
| `fastfetch` | System info config + 30-image Star Wars ASCII art gallery |
| `ghostty` | Primary terminal config |
| `git` | Global gitignore |
| `karabiner` | Karabiner-Elements key remapping |
| `kitty` | Kitty terminal config |
| `mpd` | Music Player Daemon config (CoreAudio output) |
| `raycast` | Custom Raycast scripts |
| `rmpc` | TUI MPD client config + gruvbox-material theme |
| `shell` | `.zprofile`, `.tool-versions`, `.hushlogin` |
| `tmux` | Tmux config |
| `vscodium` | VSCodium profile |
| `yazi` | File manager config + gruvbox-dark flavor |
| `zsh` | `.zshrc` + `aliases.zsh` |

## Fresh Install

```bash
# Clone
git clone git@github.com:cfsanderson/mac-dotfiles.git ~/Projects/mac-dotfiles

# Run install script (installs Homebrew if needed, stows all packages, runs brew bundle)
cd ~/Projects/mac-dotfiles && ./install.sh
```

### Post-install

- **Shell:** Open a new terminal — zsh + oh-my-zsh should load with all aliases
- **Neovim:** Run `nvim` — lazy.nvim will install all plugins automatically
- **Music:** Run `music` to open rmpc; MPD runs as a launchd service (starts at login)
- **Karabiner:** Open Karabiner-Elements to activate key remapping profiles
- **gh:** Run `gh auth login` to authenticate (tokens are not tracked)

### Not tracked (must set up manually)

- `~/.config/gh/hosts.yml` — regenerated by `gh auth login`
- `~/.config/zsh/.env.local` — local environment variables and secrets
- `~/.config/nvim/` — managed as a [separate repo](https://github.com/cfsanderson/kickstart-cfs.nvim)

## Color Scheme

All apps use [cfs-gruvbox-material](https://github.com/cfsanderson/cfs-gruvbox-material), a fork of [Sainnhe's Gruvbox Material](https://github.com/sainnhe/gruvbox-material). The master palette lives in `colors/` with CSS, TOML, and shell variable exports for consistent theming across tools.
